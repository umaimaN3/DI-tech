# 🌐 DI Tech Solutions – Business Website

A modern, responsive, and professional IT services website built with **React + Vite + TypeScript**.
This project showcases a complete digital presence for an IT solutions company specializing in **Cloud Computing, Cybersecurity, and IT Infrastructure**.

---

## 🚀 Overview

**DI Tech Solutions** empowers businesses with secure, scalable, and efficient technology solutions including:

* ☁️ Cloud Services (Microsoft 365)
* 🔐 Cybersecurity (Trend Micro, Kaspersky)
* 🖥️ IT Infrastructure & Virtualization
* 📊 Managed IT Services

## ⚙️ Getting Started

Clone the repository:

```bash
git clone https://github.com/umaimaN3/DI-tech.git
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```
